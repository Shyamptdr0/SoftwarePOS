# Application Assets

This directory contains application icons and assets for the Shreem POS desktop application.

## Required Icons

- `icon.ico` - Windows icon file (256x256px)
- `icon.icns` - macOS icon file (512x512px)  
- `icon.png` - Linux icon file (512x512px)

## Instructions

1. Create or obtain application icons in the appropriate formats
2. Place them in this directory with the exact filenames specified above
3. The electron-builder configuration will automatically pick up these files during build

## Icon Requirements

- Use the Shreem POS branding colors and logo
- Ensure high resolution for crisp display on all platforms
- Follow platform-specific guidelines for icon design
